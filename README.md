# BankApi

API REST desenvolvida em Java com Spring Boot simulando funcionalidades básicas de um sistema bancário.

## Funcionalidades
- Autenticação com JWT
- Cadastro e login de usuários
- Criação de contas bancárias
- Depósito em conta
- Registro de transações

## Tecnologias
- Java 17+
- Spring Boot
- Spring Security
- JWT
- JPA / Hibernate
- PostgreSQL / H2

Projeto desenvolvido para fins de estudo e prática em backend.
